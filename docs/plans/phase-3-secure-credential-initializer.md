## Phase B — MSBuild Code Generation Target

### Changes to `LifeGrid.Infrastructure.csproj`

**1. Path properties** (inside a `<PropertyGroup>` after the existing one):

```xml
<PropertyGroup>
  <_GeminiKeyFile>$([MSBuild]::NormalizePath(
    '$(MSBuildProjectDirectory)', '..', '..', 'z-ai-com', 'gemini.api.key'
  ))</_GeminiKeyFile>
  <_GeneratedProviderFile>$(MSBuildProjectDirectory)\obj\BuildSecretProvider.g.cs</_GeneratedProviderFile>
</PropertyGroup>
```

**2. BeforeBuild target** (inline C# task via `RoslynCodeTaskFactory`):

```xml
<Target Name="GenerateBuildSecretProvider" BeforeTargets="BeforeBuild">

  <!-- Hard-fail: file missing -->
  <Error
    Condition="!Exists('$(_GeminiKeyFile)')"
    Text="Gemini API key file not found at $(_GeminiKeyFile). Run scripts\setup-dev-secrets.ps1." />

  <ReadLinesFromFile File="$(_GeminiKeyFile)">
    <Output TaskParameter="Lines" ItemName="_GeminiKeyLines" />
  </ReadLinesFromFile>

  <!-- Hard-fail: file empty -->
  <Error
    Condition="'@(_GeminiKeyLines)' == ''"
    Text="Gemini API key file is empty. Run scripts\setup-dev-secrets.ps1." />

  <!-- Generate BuildSecretProvider.g.cs -->
  <GenerateBuildSecretProviderTask
    ApiKey="@(_GeminiKeyLines)"
    OutputPath="$(_GeneratedProviderFile)" />

  <!-- Include generated file in compilation -->
  <ItemGroup>
    <Compile Include="$(_GeneratedProviderFile)" />
  </ItemGroup>

</Target>

<UsingTask
  TaskName="GenerateBuildSecretProviderTask"
  TaskFactory="RoslynCodeTaskFactory"
  AssemblyFile="$(MSBuildToolsPath)\Microsoft.Build.Tasks.Core.dll">
  <ParameterGroup>
    <ApiKey ParameterType="System.String" Required="true" />
    <OutputPath ParameterType="System.String" Required="true" />
  </ParameterGroup>
  <Task>
    <Using Namespace="System" />
    <Using Namespace="System.IO" />
    <Using Namespace="System.Text" />
    <Code Type="Fragment" Language="cs"><![CDATA[
      const byte XorSalt = 0xAB;
      byte[] raw = Encoding.UTF8.GetBytes(ApiKey.Trim());
      var sb = new StringBuilder();
      for (int i = 0; i < raw.Length; i++)
      {
          if (i > 0) sb.Append(", ");
          sb.Append((byte)(raw[i] ^ XorSalt));
      }
      string content =
          "// <auto-generated/>\n" +
          "namespace LifeGrid.Infrastructure.Security;\n" +
          "internal sealed partial class BuildSecretProvider : IBuildSecretProvider\n" +
          "{\n" +
          $"    private static readonly byte[] _s = new byte[] {{ {sb} }};\n" +
          "    public byte[] GetObfuscatedBytes() => (byte[])_s.Clone();\n" +
          "    public byte GetXorSalt() => 0xAB;\n" +
          "}\n";
      Directory.CreateDirectory(Path.GetDirectoryName(OutputPath)!);
      File.WriteAllText(OutputPath, content, Encoding.UTF8);
    ]]></Code>
  </Task>
</UsingTask>
```

**Note:** `obj\` is already covered by `[Oo]bj/` in `.gitignore` — no additional git rule needed.

### MAUI package reference

`LifeGrid.Infrastructure` does not have `UseMaui=true`, so `$(MauiVersion)` will be undefined without extra wiring. **A new `Directory.Build.props` at the solution root** defines it:

```xml

  <PropertyGroup>

    <!-- Verify with: dotnet workload list -->
    <MauiVersion>10.0.100</MauiVersion>
  </PropertyGroup>
</Project>
```

*(Exact version confirmed during implementation with `dotnet workload list` or by inspecting Presentation's restored packages.)*

`SecureStorageService` implementation sketch:
```csharp
internal sealed class SecureStorageService : ISecureStorageService
{
    public Task<string?> GetAsync(string key) =>
        Microsoft.Maui.Storage.SecureStorage.GetAsync(key);
    public Task SetAsync(string key, string value) =>
        Microsoft.Maui.Storage.SecureStorage.SetAsync(key, value);
}
```

`ApiCredentialSyncService` logic (in full):
```
SyncAsync():
  existing = await _secureStorage.GetAsync("Gemini_Provider_Token")
  if existing != null → return                          // Case A

  obfuscated = _buildSecretProvider.GetObfuscatedBytes()
  salt       = _buildSecretProvider.GetXorSalt()
  decoded    = new byte[obfuscated.Length]
  for i: decoded[i] = obfuscated[i] XOR salt
  plaintext  = UTF8.GetString(decoded)
  await _secureStorage.SetAsync("Gemini_Provider_Token", plaintext)
  Array.Clear(decoded, 0, decoded.Length)               // zero-out
  decoded    = null
```

## Phase F — Developer Setup Script

### New file: `scripts\setup-dev-secrets.ps1`

Steps the script performs:
1. Verify `dotnet` is on `$PATH` — exit with error if not.
2. Resolve repo root (parent of the script's directory).
3. Create `z-ai-com\` directory at repo root if absent.
4. Prompt: `Read-Host -AsSecureString "Enter your Gemini API key"` → convert to plaintext string.
5. Write the key to `z-ai-com\gemini.api.key` (UTF-8, no BOM, single line, no trailing newline).
6. Run `dotnet build LifeGrid.slnx` from the repo root.
7. Print `SUCCESS` if exit code is 0; otherwise print `BUILD FAILED — check the output above` and exit with code 1.
